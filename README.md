# AI Assistant Product Case

Продуктовый кейс разработки AI-ассистента для пользователей с СДВГ.

Ассистент адаптирует рекомендации в зависимости от состояния пользователя:

- уровень энергии
- эмоциональное состояние
- доступное время
- текущий этап выполнения задачи

Цель продукта — помочь пользователям продолжать выполнение задач и снижать когнитивную нагрузку.
